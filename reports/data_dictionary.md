# Data Dictionary

## nav_history

| Column | Description |
|----------|-------------|
| amfi_code | Unique fund code |
| scheme_name | Mutual fund name |
| nav | Net Asset Value |
| date | NAV date |

## fund_performance

| Column | Description |
|----------|-------------|
| start_nav | Starting NAV |
| end_nav | Ending NAV |
| return_pct | Percentage return |

## risk_analysis

| Column | Description |
|----------|-------------|
| volatility | Standard deviation of returns |
| sharpe_ratio | Risk-adjusted return |
